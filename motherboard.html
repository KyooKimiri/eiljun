Motherboard

The motherboard is the main circuit board of a computer, serving as the backbone that connects all other components, such as the CPU, RAM, storage, and peripherals. It facilitates communication between these components, allowing the computer to function as a unified system.

Key Components of the Motherboard

  • CPU Socket: The slot where the CPU is installed.
  • RAM Slots: Slots for installing system memory (RAM).
  • Chipset: Manages data flow between the processor, memory, and peripherals.
  • Expansion Slots: Slots for adding extra components, like graphics or sound cards.
  • Power Connectors: Connectors that supply power to the motherboard and components.
